# 🎮 YunDoom

**YunDoom** è un gioco sviluppato in Python, progettato per essere semplice da eseguire, ma anche completamente aperto alla modifica e alla personalizzazione. Il progetto è **open source**, quindi chiunque può esplorarne il codice, modificarlo e contribuire con miglioramenti o nuove funzionalità.

---

## 🚀 Caratteristiche

* Sviluppato interamente in Python
* Basato su Pygame
* Open Source: modificabile liberamente
* Struttura semplice e facilmente estendibile

---

## 📦 Requisiti

Prima di eseguire YunDoom, assicurati di avere:

* Python 3.x installato
* `pygame` installato

---

## 🛠️ Installazione

1. **Scarica il progetto**

   * Clona la repository oppure scarica lo ZIP:

   ```bash
   git clone <URL_DEL_REPOSITORY>
   cd YunDoom
   ```

2. **(Consigliato) Crea un ambiente virtuale**

   Usare un ambiente virtuale aiuta a mantenere isolate le dipendenze del progetto:

   ```bash
   python -m venv venv
   ```

   Attivalo:

   * Su Windows:

     ```bash
     venv\Scripts\activate
     ```
   * Su macOS/Linux:

     ```bash
     source venv/bin/activate
     ```

3. **Installa Pygame**

   ```bash
   pip install pygame
   ```

---

## ▶️ Avvio del gioco

Per avviare YunDoom, esegui:

```bash
python main.py
```
